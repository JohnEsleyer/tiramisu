# Examples

The `examples/` folder contains runnable demos and reference implementations.

## Running The Example Server

```bash
pnpm run dev
```

Then open `http://localhost:3001` to browse the list of examples.

## Notable Examples

- `examples/music-visualizer`: audio-reactive visuals built on the 2D canvas API.
- `examples/meme-generator`: drag-and-drop meme builder with text overlays.
- `examples/snow-overlay`: deterministic snow particles using `TiramisuUtils`.
- `examples/video-editor`: basic clip-based compositor.
- `examples/webgl-editor`: WebGL/WebCodecs-based editor UI.
- `examples/split-screen`: side-by-side layout with fit/cover helpers.
- `examples/video-audio-overlay`: audio-reactive overlays on video.

Use these as templates for composing clips, driving animations, and integrating assets or video sources.
